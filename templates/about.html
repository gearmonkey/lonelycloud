About the loneliest cloud



What is the sound of loneliness? What strange acoustic binds tracks which are not even listened to by their creator/uploader, sometimes for years?
The Loneliest Cloud is an attempt to explore the unfound and unplayed on Soundcloud.
In so doing, it surfaces the strange, the forgotten, the sometimes beautiful.
All the tracks the sytem presents have a playcount of zero on soundcloud. As they are autoplayed on load, the system will only ever present a track once. In this way we hear the forgotten, but only for a moment.

Technically, the system has to go looking for unplayed soundcloud tracks, as there is no means to directly search (via the soundcloud api) for track with a playcount filter or order. To do this and still have a reasonably responsive service, a simple async worker queue is set up using rq and redis.
The system is a python flask web app, deployed on heroku. The source code is on github.
